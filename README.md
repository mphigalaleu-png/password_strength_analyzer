# password_strength_analyzer
A Python application that evaluates password strength based on:

- Length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## Features

- Strength scoring
- Security recommendations
- Easy-to-use interface

## Installation

git clone https://github.com/mphigalaleu-png/password_strength_analyzer.git

cd password-strength-analyzer

python password_strength analyzer.py

## Technologies Used

- Python 3

## Future Improvements

- GUI version
- Password generator
- Password breach checking

Password: P@ssword123

Strenth: Strong
